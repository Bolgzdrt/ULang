# ULang

## Project setup

```sh
npm install
```

## Runs Development Environment

```sh
npm run start
```

or

```sh
npm run start:dev
```

### Compiles and hot-reloads front-end for development

```sh
npm run serve
```

### Compiles and minifies for production

```sh
npm run build
```

### Run unit tests

```sh
npm run test:unit
```
